<summary><b>🤖 GovBot</b> &nbsp;·&nbsp; WhatsApp-first AI assistant for Indian government services &nbsp;&nbsp;<code>● LIVE</code></summary>
<br/>

**GovBot** is an AI-powered public service assistant built by **Shashank Gowda / shashank03-dev** to simplify access to Indian government schemes and services through WhatsApp and a web dashboard.

**The problem:** Citizens often struggle to navigate government portals, eligibility rules, scheme documents, and application steps.

**What it does:** A user sends a WhatsApp message in plain language. A **LangGraph multi-agent system** handles intent routing, uses a **RAG pipeline** over scraped government documents, and returns a clear, actionable answer. The project also includes a **Next.js operator dashboard** for monitoring sessions and managing the knowledge base.

| Layer | Technology |
|---|---|
| Agent orchestration | LangGraph |
| API backend | FastAPI |
| Vector store | ChromaDB |
| Web scraping | Playwright |
| Database | Supabase |
| Frontend | Next.js 15 |

| Deployment | Railway + Vercel |
