I build AI agents that ordinary people can actually use, and I like understanding the layer underneath them well enough to rebuild it. That habit is why there's a relational database written in plain C sitting in my repos. I work out of Bengaluru, and most of what I make points at problems I can see around me in India.

<br/>

### Now

- **GovBot** is live. You ask an Indian government question on WhatsApp, LangGraph routes the intent, and a RAG pipeline over scraped official documents answers it in plain language.
- **MotionCode** is live. Drop in a short clip of a UI animation and it hands back the GSAP or Framer Motion code for that exact motion.
- On the side: working through NVIDIA's deep-learning track, and reading storage-engine internals because the database itch never fully went away.
